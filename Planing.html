HOJAS DE ESTILO CC INTERNAS

¿Que es CSS?

* Significa Cascading Style Sheets ( hoja de estilos con cascada).

*El lenguaje CSS describe como se mostraran los elementos HTML en la pantalla.

*El lenguaje CSS ahorra mucho trabajo, ya que permite controlar el diseño de varias paginas web a la vez.

*Todas las hojas del estilo externas se almacenan en archivos CSS.

¿Por que usar CSS?

El lenguaje CSS se utiliza para definir los estilos de tus paginas web, incluyendo el diseño, la disposicion y las variaciones en la imagen para los diferentes dispositivos y tamaños de pantalla.

Tres formas de insertar CSS...

*Hoja de estilos interna

*Hoja de estilos externa

*Estilo de linea

HOJA DE ESTILOS EXTERNA:Con una hoja de estilos externa puedes cambiar el aspecto de un sitio web completo cambiando un solo archivo. Para ello, cada pagina debe incluir una referencia al archivo de hoja de estilos externa dentro del elemento que se encuentra dentro de la seccion "chead".

Una hoja de estilo externo se puede describir desde cualquier editor de texto, debe ser guardado con la extension.css y no puede contener ninguna etiqueta HTML.

HOJA DE ESTILOS INTERNA:Una hoja de estilos interna se puede usar en una sola pagina con un estilo union. Los estilos internos se definen dentro del elemento "style" dentro de la propia pagina HTML.

ESTILOS EN LINEA:Un estilo en linea se puede utilizar aplicandolo al estilo de un solo elemento. Para ello tienes que añadir el atributo de estilo al elemento en cuestion pudiendo contener cualquier propiedad CSS.

HOJAS DE ESTILO CCS EXTERNAS

Para definir el estilo de una pagina de una web desde un archi externo, debe crearse un archivo de texto, conteniendo las definiciones y guardarlo con la extencion *css. Como cualquier link, la localizacion del archivo CSS puede ser absoluta o relativa, dependiendo de su ubicacion dentro del propio sitios web o en otra direccion.

El link siempre debe insertarse dentro de las etiquetas HEAD del ancabezamiento de la pagina web.

SELECTORES CSS

¿Que son?

Es el identificador en la hoja de estilos del elemento o elementos del DOM a los que se aplicaran las declaraciones contenidas en la regla de la que forma parte dicho selector.

encontramos tres tipos de selectores...

* Selector CSS simple

* Selector CSS compuesto

* Selector CSS complejo

Veremos la definicion de cada uno de ellos:

SELECTOR CSS SIMPLE: Es aquel que esta formado solo por una unica cadena textual, sin nungun combinador. Son selectores simples el sector universad *.

SELECTOR CSS COMPUESTO: Es una cadena de selectores simples sin combinadores (el espacio en blanco tambien queda excluido porque es un combinador): section.noticias{} .una_clase.otra_clase{}.

SECTOR CSS COMPLEJO: Es una secuencia o cadena de selectores separados por "combinadores"
